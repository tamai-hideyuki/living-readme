# Parts

This folder stores small, reusable SVG snippets that can be copied into `src/defs/*.svg` and `src/bodies/*.svg`.

## defs
- `defs/gradients.defs.svg`: common gradients
- `defs/filters.defs.svg`: common blur filters

## bodies
- `bodies/backgrounds.svg`: background rect variants
- `bodies/orbs.svg`: soft orb cluster
- `bodies/waves.svg`: wave/flow lines
- `bodies/particles.svg`: floating particles
- `bodies/geometry.svg`: geometric cluster

## Usage
Copy the required snippets into a scene's `defs` or `body` file. Keep IDs unique per SVG.
